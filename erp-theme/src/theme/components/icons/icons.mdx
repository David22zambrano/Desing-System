# ICONS ANGULAR MATERIAL

Los iconos de Angular Material son visualmente simples y limpios, y utilizan una paleta de colores monocromáticos.
Algunos de los iconos más comunes incluyen:

<ul>
  <li>Flechas de dirección para indicar la navegación</li>
  <li>
    Iconos de menú y hamburguesas para indicar la existencia de un menú
    desplegable
  </li>
  <li>Iconos de búsqueda para indicar la presencia de una barra de búsqueda</li>
  <li>Iconos de lápiz para indicar la capacidad de editar un elemento</li>
  <li>
    Iconos de papelera para indicar la capacidad de eliminar un elemento Iconos
    de
  </li>
  <li>cerrar para indicar la capacidad de cerrar una ventana o modal.</li>
</ul>
En general, los iconos de Angular Material son diseñados para ser claros y concisos,
y para mejorar la usabilidad de las aplicaciones web y móviles construidas con Angular.
