# Avatar

Los avatares de Angular Material son una imagen o icono pequeño que se puede utilizar para representar a un usuario o entidad en una aplicación web de Angular. Son personalizables con diferentes tamaños, formas, colores y estilos, y son útiles para mejorar la identificación visual de los usuarios en la interfaz de usuario.
